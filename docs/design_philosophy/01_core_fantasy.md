# Core Fantasy & Player Psychology

## The God-Complex Engineering
Project SL-OMEGA không chỉ là game adaptation - đây là psychological engine được thiết kế để maximize LTV, DAU, và ARPPU.

### Core Progression Path
1. **Mortal** → Weak, insignificant
2. **Spirit Master** → Awakened potential  
3. **Spirit Elder/Ancestor** → Regional power
4. **Titled Douluo** → Server legend
5. **Limit Douluo** → Server ruler
6. **God** → Transcendent entity

### Four Pillars of Satisfaction
- **Power Fantasy**: Damage numbers 150 → 1,500,000,000
- **Dominance Fantasy**: One-shot kills, unkillable in PvP
- **Prestige Fantasy**: Exclusive titles, auras, chat colors
- **Scarcity Psychology**: Limited drops maintain value

## Player Journey Mapping

### Day 1 (The Hook)
- 0-30min: High-octane tutorial, instant rewards
- 30-60min: First friction + $0.99 starter pack trigger
- End: Guild join requirement

### Week 1 (Habit Formation)
- Days 2-3: Gacha introduction with rigged beginner luck
- Days 4-5: First upgrade failure + protection stone upsell
- Day 7: 7-day login climax reward

### Month 1+ (Sunk Cost Trap)
- Level cap reached → Gear optimization focus
- PvP introduction creates spending motivation
- Social guild lock-in prevents churn
